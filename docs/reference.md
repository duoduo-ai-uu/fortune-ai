## 二、功能参考

### 1. 算命类型
- 🔮 综合运势 - 整体情况
- 💕 爱情运程 - 桃花、感情
- 📈 事业发展 - 工作、升职
- 💰 财富运势 - 投资、财运
- 🌿 健康平安 - 身体、平安

### 2. 交互模式
- 选择类型 → 输入问题 → AI 回复
- 打字机效果展示回复
- 消息气泡区分用户/AI
